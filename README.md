# 🥤 SODA Project
1. 프로젝트명 : 협업툴 - SODA
2. 설명 : 협업툴 Flow 를 참고하여 만든 협업툴 프로젝트 사이트 입니다.
3. 소개 : COVID-19로 인한 비대면 업무가 중요해진 시대에서 장소와 시간의 제약을 받지 않고 업무를 볼 수 있다는 장점을 살려, <br>
   회사 뿐만 아니라 학교와 병원 등에서 다양하게 사용할 수 있도록 만든 협업툴 입니다.
4. 개발기간 : 2023.08.28 ~ 2024.02.07
